### BrainStormer - SenchaCON 2013 Hackathon Meeting Notes App!
=============

### Overview
Built in 3 days (and 3 nights pretty much non-stop) for the SenchaCON 2013 Hackathon, BrainStormer is a whiteboard meeting app that lets you
